# Convex

**Def:** A set $\mathcal{S} \subseteq \mathbb{R}^d$ is __convex__ iff $\forall s_1, s_2 \in \mathcal{S}$, the line segment $\overbar{s_1 s_2}$ is also contained in $\mathcal{S}$.
